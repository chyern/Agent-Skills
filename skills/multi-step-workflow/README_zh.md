# 多步骤工作流 (自适应 SOP + 自主循环)

轻量级 AI Agent 任务追踪工具。该技能为复杂工程任务强制执行专业的 **Planning Mode (规划模式)**，并在此之后通过 **Autonomous Loop (自主循环)** 高速推进任务。

## 自适应工作流逻辑

Agent 根据任务复杂度进行分流，以最大化效率：

1. **快速路径 (Simple Path < 3 步)**：针对简单的单项任务（如读取文件、解释函数）。Agent 直接执行。
2. **标准路径 (Standard Path >= 3 步)**：针对工程任务。
   - **第 1 步：规划模式**：Agent 展示计划，并**必须停止以等待您的批准**。
   - **第 2 步：自主循环**：一旦计划获批，Agent 进入自主执行周期——它完成一个步骤，简要汇报进展，然后**立即开始下一个步骤**，无需等待您的回复。

## 为什么需要

本技能解决了“AI 偷懒”或“AI 频繁中断”的问题。通过在您批准计划后授予 Agent“自主执行权”，它可以在一次对话中完成整项工作，同时仍然让您保留了在任务动工前审查方案的控制权。

## 安全与 ClawHub 说明

> [!IMPORTANT]
> **为什么使用 `always: true`?**
> 本技能提供了一套标准作业程序 (SOP)。通过设置 `always: true`，可以让 Agent 始终意识到应对复杂任务时*必须*先进行规划。
>
> **自主执行说明**
> “自主循环 (Autonomous Loop)”仅在您明确批准了实施计划之后才会触发。您保留了对自动化开启点的完全控制。

## 标准用法 (>= 3 步)

1. **分析流程**：Agent 识别任务为“标准路径”。
2. **规划阶段**：Agent 创建步骤并拟定实施计划 (implementation_plan.md)。
3. **审批阶段**：Agent 会说“我正处于规划模式”并**停止 (STOP)**。
4. **自主循环**：一旦您说“OK”，Agent 将循环执行所有步骤，每步完成后汇报，直到全部完成。

## 手动命令 (可选)

```bash
# 查看所有活动任务和步骤
node scripts/task-tracker.js list

# 标记特定步骤完成
node scripts/task-tracker.js done "任务名" 1
```
